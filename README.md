# Processamento de Imagem

Descrição: 
The package image_processing is used to:
	- Match histogram
	- Structural similarity
	- Resize image
	- Plot image
	- Plot histogram
	- Plot results
	- Save image
	- Read image

## Instalação

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install package_name

```bash
pip install image_processing
```
